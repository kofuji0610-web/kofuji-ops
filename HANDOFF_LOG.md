## 現在の実装状態（2026-04-29 時点）

### メインファイル

### 実装済み機能一覧

#### 整備日報フォーム
- **車両別整備記録**：複数車両対応、車両ごとに追加・削除可能
- **整備部位カテゴリ**：ボタン式セレクター（エンジン系・動力伝達系・足回り・ブレーキ・電気・ボディ・その他）
- **点検項目**：カテゴリごとのチェックボックス選択
- **整備内容**：点検項目ごとにインライン表示
  - 異常なし チェックボックス（チェックするとグレーアウト）
  - 状態・処置 ドロップダウン（点検項目ごとに適切な選択肢のみ表示）
  - 測定値入力（ブレーキパッド残厚 等）
  - 交換・補充部品（数量・取付位置）
- **タイヤローテーション**：FL/FR/RL/RR の回転先を個別選択できる専用UI
- **グリスアップ**：足回り・動力伝達系カテゴリに追加、部品サジェスト（グリス/g）
- **作業時間**：開始・終了時刻、合計時間バナー表示
- **作業終了ボタン**（緑）：車両総合判定の下に配置
- **完了後確認チェック**：エンジン始動・試走・漏れなし・灯火確認
- **車両総合判定**：異常なし/要観察/要修理/緊急対応
- **要注意 → 運行管理へ報告** フラグ

#### 複数車両対応
- 車両を追加ボタン（案内コメント付き）
- 作業時間サマリー：各車両の時間 + 全車両合計時間（2台以上で表示）

#### 日報提出
- **日報を提出するボタン**（緑・全フロー共通）：ページ最下部
- **下書き保存**ボタン併設
- **提出完了画面**：「本日の業務お疲れさまでした。からだに気をつけてゆっくり休んでください。」

#### 休憩ボタン（フローティング）
- 画面右側に固定表示（`fixed right-4 top-1/2`）
- ☕ 休憩開始 → 現在時刻を記録
- ✅ 休憩終了 → 現在時刻を記録
- 完了後は開始〜終了時刻をコンパクト表示

#### 業務内容ブロック
- 業務１：部署選択（整備の場合は案内テキスト、他部署は内容入力テキストエリア）
- 業務２以降：（仮）表示 + テキストエリア
- 業務を追加ボタン

## 点検項目ごとの状態・処置マップ（ITEM_CONDITION_MAP / ITEM_ACTION_MAP）

プロ整備士の知識に基づき各点検項目に適切な選択肢を設定済み：

| 点検項目 | 状態の選択肢 | 処置の選択肢 |
|---|---|---|
| エンジンオイル | 正常/劣化/漏れあり/その他 | 点検のみ/部品交換/要観察 |
| タイヤ摩耗 | 正常/摩耗あり/偏摩耗/損傷/その他 | 点検のみ/部品交換/要観察 |
| タイヤローテーション | 正常/その他 | 位置調整/点検のみ |
| ハブ・ベアリング | 正常/摩耗あり/異音/その他 | 点検のみ/給脂・注油/部品交換/要観察 |
| グリスアップ | 正常/摩耗あり/その他 | 給脂・注油/点検のみ |
| ブレーキパッド | 正常/摩耗あり/要交換/その他 | 点検のみ/部品交換 |
| （その他多数） | ... | ... |
